## 1. 정의

사용자·엔티티를 시각적으로 식별하는 원형 이미지·이니셜 표시. 댓글·프로필·멤버 리스트에서 사용.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `root` | ✓ | 원형 컨테이너 |
| `image` | – | 사진 |
| `initials` | – | 이니셜 폴백 |
| `presence` | – | 온라인 상태 점 |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| size | `sm` (24) \| `md` (32) \| `lg` (48) \| `xl` (64) |
| tone | `muted` \| `primary` |

## 4. 토큰 매핑 — tone: muted, size: md

| 속성 | 토큰 | 값 |
|---|---|---|
| background | `{alias.surface.muted}` | {map.color.neutral.50} |

## 5. 사용 케이스 (Do)

- 댓글·멤버 리스트의 사용자 식별
- 이미지 없을 때 이니셜 폴백 (한국 성씨 1자 권장)

## 6. 사용 금지 (Don't)

- 의미 없는 장식 (불필요한 시각 노이즈)
- 텍스트 라벨 없이 단독 사용 (스크린리더 의미 전달 부족)

## 7. 접근성

- `<img alt="...">` 필수, 이니셜은 `aria-label`
- 이미지 로드 실패 시 이니셜로 폴백
- 충분한 대비 — 이니셜 텍스트 × 배경 ≥ 4.5:1

## 8. 코드 예시

```html
<span class="xds-avatar" aria-label="홍길동">홍</span>
```
