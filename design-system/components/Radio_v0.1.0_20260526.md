## 1. 정의

한 그룹에서 정확히 1개를 선택하는 폼 컨트롤. 항상 같은 `name` 그룹 안에서 사용.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `circle` | ✓ | 원형 외곽 |
| `dot` | ✓ | 선택 시 표시되는 안쪽 점 |
| `label` | ✓ | 텍스트 라벨 |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| state | `unchecked` \| `checked` \| `disabled` |

## 4. 토큰 매핑 — state: checked

| 속성 | 토큰 | 값 |
|---|---|---|
| circle.border | `{alias.action.primary}` | {seed.color.primary} |
| dot.background | `{alias.action.primary}` | {seed.color.primary} |
| circle.background | `{alias.surface.elevated}` | oklch(1 0 0) |

## 5. 사용 케이스 (Do)

- 2~7개 옵션 중 1개 선택 (성별·결제 수단 등)
- 옵션 수 8개 초과 시 `Select`로 대체

## 6. 사용 금지 (Don't)

- 다중 선택 → `Checkbox`
- 그룹 라벨 없이 단독 사용

## 7. 접근성

- `<fieldset>` + `<legend>`로 그룹 라벨 (또는 `role="radiogroup"` + `aria-label`)
- 키보드: 그룹 진입 후 ↑↓ 또는 ←→ 로 이동
- 같은 `name` 속성으로 그룹화 (Tab은 그룹 단위로 1회)

## 8. 코드 예시

```html
<fieldset>
  <legend>결제 방법</legend>
  <label><input type="radio" name="pay" value="card" /> 카드</label>
  <label><input type="radio" name="pay" value="bank" /> 계좌이체</label>
</fieldset>
```

## 9. 관련 컴포넌트

- Checkbox — 다중 선택
- Select — 8개 초과 옵션
- SegmentedControl — 시각적으로 압축된 라디오
