## 1. 정의

사용자가 클릭으로 액션을 트리거하는 기본 컨트롤.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `root` | ✓ | 버튼 컨테이너 |
| `label` | ✓ | 텍스트 |
| `iconLeading` | – | 좌측 아이콘 |
| `iconTrailing` | – | 우측 아이콘 |
| `spinner` | – | loading 상태 표시 |

## 5. 사용 케이스 (Do)

- 한 화면의 주요 액션 1개에 `primary` 사용
- 위험 액션(삭제·취소불가)에 `danger`
- 보조 액션에 `default` 또는 `dashed`

## 6. 사용 금지 (Don't)

- 한 화면에 `primary` 버튼 여러 개
- 라벨 6단어 초과
- 아이콘만 있는 버튼 (별도 `IconButton` 컴포넌트 사용)
