## 1. 정의

0개 이상 다중 선택이 가능한 폼 컨트롤. 부분 선택(indeterminate) 상태 지원.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `box` | ✓ | 사각 박스 |
| `check` | ✓ | 체크/대시 아이콘 |
| `label` | ✓ | 텍스트 라벨 |

## 3. Variants

| 차원 | 옵션 |

## 5. 사용 케이스 (Do)

- 다중 선택 (예: 약관 동의 항목 여러 개)
- 트리 구조의 부분 선택 (`indeterminate` 활용)

## 6. 사용 금지 (Don't)

- 단독 ON/OFF → `Switch`
- 1개만 선택 → `Radio`

## 7. 접근성

- `<label>` 또는 `aria-label` 필수 (U-03)
- `indeterminate`는 `aria-checked="mixed"`로 표현
- 키보드: Space로 토글, Tab 순서 자연스럽게
- 색만으로 상태 구별 금지 (P-06)

## 8. 코드 예시

```html
<label class="xds-checkbox">
  <input type="checkbox" />
  <span>약관에 동의합니다</span>
</label>
```

## 9. 관련 컴포넌트

- Radio — 1개만 선택
- Switch — 즉시 반영 토글
