## 1. 정의

데이터·검색 결과가 *0건일 때* 표시하는 안내 상태. 단순 "없음" 알림보다 *다음 행동 유도*를 포함한다.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `illustration` | – | 아이콘·간단 일러스트 |
| `title` | ✓ | 짧은 상태 설명 |
| `description` | – | 보조 설명 |
| `action` | – | 다음 행동 버튼 ("새로 만들기") |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| context | `no-data` \| `no-result` \| `error` \| `first-time` |

## 5. 사용 케이스 (Do)

- 첫 사용자 빈 대시보드 ("새 프로젝트로 시작해보세요")
- 검색 결과 0건 ("다른 키워드로 검색해보세요")
- 다음 행동 명확한 1개 CTA 동반

## 6. 사용 금지 (Don't)

- 단순 "데이터 없음" 끝 (행동 유도 없음)
- 오류 상태와 혼동 → 별도 `Result` 컴포넌트
- 인터랙티브 요소 가리는 위치 (모달 위 등)

## 7. 접근성

- 일러스트는 `aria-hidden` (장식)
- 제목 `<h2>` 또는 `<h3>` (페이지 구조와 일관)
- CTA는 명확한 동사 ("새로 만들기" — "클릭"·"여기" 같은 비명시 금지, O-05)

## 8. 코드 예시

```html
<div class="xds-empty">
  <span aria-hidden>∅</span>
  <h3>표시할 항목이 없습니다</h3>
  <p>새 프로젝트를 만들어보세요.</p>
  <button class="xds-button--primary">새로 만들기</button>
</div>
```

## 9. 관련 컴포넌트

- Result — 작업 완료·실패 결과 페이지
