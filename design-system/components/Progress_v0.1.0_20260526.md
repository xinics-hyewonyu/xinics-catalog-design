## 1. 정의

결정적 진행률(0~100%)을 시각화하는 선형·원형 인디케이터. 업로드·다운로드·설치 등 *예측 가능한* 작업에 사용.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `track` | ✓ | 전체 범위 배경 |
| `fill` | ✓ | 진행률 채움 |
| `label` | – | 좌상단 작업 이름 |
| `value` | – | 우상단 "42%" |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| shape | `linear` \| `circular` |
| size | `sm` \| `md` |
| tone | `primary` \| `success` \| `warning` \| `danger` |

## 5. 사용 케이스 (Do)

- 파일 업로드·다운로드 진행률
- 단계별 작업의 전체 진행도
- 완료 후 100% → 잠시 표시 후 사라짐 또는 success tone 전환

## 6. 사용 금지 (Don't)

- 진행률 알 수 없는 비결정적 작업 → `Spinner`
- 메인 콘텐츠를 가리는 위치

## 7. 접근성

- `role="progressbar"` + `aria-valuemin/max/now`
- 텍스트 값 병기 ("42%") — 색 의존 금지 (P-06)
- 완료 시 `aria-valuenow=100` + 텍스트로도 알림
- `prefers-reduced-motion`: 진행 애니메이션 즉시 반영

## 8. 코드 예시

```html
<div role="progressbar" aria-valuemin="0" aria-valuemax="100" aria-valuenow="42">
  <span style="width: 42%"></span>
</div>
```

## 9. 관련 컴포넌트

- Spinner — 비결정적 대기
- Steps — 명시적 단계 진행
