## 1. 정의

순차적 흐름의 진행 단계를 시각화 (Stepper). 사용자가 *어느 단계에 있는지*와 *얼마나 남았는지* 표시.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `list` | ✓ | `<ol>` 단계 묶음 |
| `step.circle` | ✓ | 번호·체크 아이콘 |
| `step.label` | ✓ | 단계 이름 |
| `connector` | ✓ | 단계 사이 연결선 |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| orientation | `horizontal` \| `vertical` |
| state (per step) | `done` \| `current` \| `pending` |

## 4. 토큰 매핑

| 속성 | 토큰 | 값 |

| done.background | `{alias.action.primary}` | {seed.color.primary} |

| label.current | `{alias.text.body}` | {map.color.neutral.900} |
| label.pending | `{alias.text.caption}` | {map.color.neutral.700} |

## 5. 사용 케이스 (Do)

- 멀티 스텝 폼 (회원가입·체크아웃)
- Export·설치 마법사

## 6. 사용 금지 (Don't)

- 단계 무관 네비 → `Breadcrumb` 또는 `Tabs`
- 단계 수 7개 초과 (인지 부하)

## 7. 접근성

- `<ol aria-label="진행 단계">`
- 현재 단계 `aria-current="step"`
- 단계 번호 + 라벨 둘 다 노출 (색만 의존 금지, P-06)

## 8. 코드 예시

```html
<ol class="xds-steps" aria-label="진행 단계">
  <li>검증</li>
  <li aria-current="step">옵션</li>
  <li>처리</li>
</ol>
```

## 9. 관련 컴포넌트

- Progress — 백분율 진행
- Tabs — 같은 위계 콘텐츠 전환
