## 1. 정의

최소·최대 범위에서 단일 값을 직관적으로 선택하는 컨트롤. 볼륨·필터·임계값 등 *연속 수치*에 사용.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `track` | ✓ | 전체 범위 라인 |
| `progress` | ✓ | 현재 값까지의 채움 |
| `thumb` | ✓ | 드래그 핸들 |
| `label` | – | 좌상단 이름 |
| `value` | – | 우상단 현재 값 |
| `ticks` | – | 눈금 표시 |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| size | `sm` \| `md` |
| variant | `single` \| `range`(두 thumb) |
| state | `default` \| `disabled` |

## 4. 토큰 매핑

| 속성 | 토큰 | 값 |
|---|---|---|
| track.background | `{alias.surface.muted}` | {map.color.neutral.50} |
| progress.background | `{alias.action.primary}` | {seed.color.primary} |
| thumb.background | `{alias.surface.elevated}` | oklch(1 0 0) |
| thumb.border | `{alias.action.primary}` | {seed.color.primary} |

## 5. 사용 케이스 (Do)

- 연속·근사값 (볼륨·밝기·가격대)
- 정확한 숫자 입력이 부차적인 경우

## 6. 사용 금지 (Don't)

- 정확한 정수 입력이 우선 → `Input[type=number]`
- 옵션이 이산적 (5단계 만족도 등) → `SegmentedControl`

## 7. 접근성

- `role="slider"` + `aria-valuemin`/`max`/`now`
- 키보드: ←→ 작은 단계, PgUp/PgDn 큰 단계, Home/End 양 끝
- 값을 텍스트로도 노출 (`aria-valuetext` 또는 옆 라벨)
- thumb 터치 타겟 ≥ 24×24px (O-09)

## 8. 코드 예시

```html
<label for="vol">볼륨</label>
<input id="vol" type="range" min="0" max="100" value="40" />
```

## 9. 관련 컴포넌트

- RangeSlider — 두 thumb으로 범위 선택
- ProgressBar — 진행 표시 (입력 아님)
