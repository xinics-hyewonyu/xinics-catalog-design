## 1. 정의

별·하트 등 시각 단위로 만족도·중요도를 표시·입력하는 컨트롤. 0~5점(또는 0~10) 척도.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `list` | ✓ | 별 묶음 (`role="radiogroup"` 또는 `role="img"`) |
| `unit` | ✓ | 단일 별 (채움·반·빈) |
| `value` | – | 숫자 노출 ("3.5 / 5") |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| mode | `display`(읽기 전용) \| `input`(클릭·키보드) |
| step | `1`(정수) \| `0.5`(반 별) |
| count | `5`(기본) \| `10` |

## 5. 사용 케이스 (Do)

- 리뷰·평점 입력·표시
- 우선순위·중요도 시각화
- 숫자 텍스트 병행 노출 ("3.5 / 5")

## 6. 사용 금지 (Don't)

- 절대 수치가 중요한 데이터 ("42 / 100" 점수) → `Statistic`
- 11점 이상 척도 (인지 한계)

## 7. 접근성

- display 모드: `role="img"` + `aria-label="별점 3.5 / 5"`
- input 모드: `role="radiogroup"` + 각 별 `role="radio"` + `aria-checked`
- 키보드: ←→ 이동, Enter/Space 선택, 0 입력 가능(키보드)
- 색만으로 채움·빈 구별 금지 (P-06) — 시각 패턴(채움 vs 윤곽) 병행
- 별 자체는 `aria-hidden`, 의미는 컨테이너 라벨에

## 8. 코드 예시

```html
<span role="img" aria-label="별점 3.5 / 5">
  <span aria-hidden>★★★½☆</span> 3.5
</span>
```

## 9. 관련 컴포넌트

- Slider — 연속 수치 입력
- Statistic — 정량 단일 값 표시
