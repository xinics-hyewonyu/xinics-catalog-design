## 1. 정의

달력 그리드에서 날짜를 선택하는 폼 컨트롤. 입력 필드 + 팝업 캘린더 형태.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `input` | ✓ | 텍스트 입력 (YYYY-MM-DD) |
| `trigger` | – | 캘린더 아이콘 |
| `calendar` | ✓ | 월별 그리드 팝업 |
| `header` | ✓ | 월·년 표시 + 좌우 이동 |
| `weekdays` | ✓ | 일~토 라벨 |
| `day` | ✓ | 개별 날짜 |

## 5. 사용 케이스 (Do)

- 예약·기간 선택 화면
- 한국식 날짜 표기(YYYY-MM-DD) 우선

## 6. 사용 금지 (Don't)

- 미래·과거 제약 없이 자유 입력만 → `Input[type=date]`로 충분
- 시간까지 필요 → `DateTimePicker`

## 7. 접근성

- `role="dialog"` + `aria-label="날짜 선택"`
- 날짜 그리드: `role="grid"` + 키보드 ↑↓←→ 이동
- 오늘 날짜: `aria-current="date"`
- 선택된 날짜: `aria-pressed="true"` 또는 `aria-selected="true"`
- 입력 필드 형식 사전 안내 (U-13) — "YYYY-MM-DD"

## 8. 코드 예시

```html
<label>
  <span>예약 날짜</span>
  <input type="text" placeholder="YYYY-MM-DD" aria-describedby="dp-help" />
  <span id="dp-help">형식: 2026-05-18</span>
</label>
```
