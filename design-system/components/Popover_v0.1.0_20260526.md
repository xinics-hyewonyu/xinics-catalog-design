## 1. 정의

트리거 옆에 떠올라 *임의의 콘텐츠*(텍스트·폼·미니 메뉴 등)를 보여주는 부유 컨테이너. Tooltip(텍스트 전용)과 Menu(옵션 리스트)의 일반화된 버전.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `trigger` | ✓ | 클릭 대상 |
| `panel` | ✓ | 부유 콘텐츠 영역 |
| `arrow` | – | 방향 표시 삼각형 |
| `close` | – | 명시 닫기 버튼 |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| placement | `top` \| `bottom` \| `left` \| `right` (+ -start/-end) |
| trigger | `click` \| `hover`(주의: 모바일 미지원) |

## 5. 사용 케이스 (Do)

- 단축키 안내·도움말
- 인라인 미니 폼 (이름 바꾸기 등)
- 컨텍스트 액션 모음

## 6. 사용 금지 (Don't)

- 텍스트만이면 → `Tooltip`
- 옵션 리스트만이면 → `Menu`
- 차단 결정이 필요한 메시지 → `Modal`

## 7. 접근성

- trigger: `aria-haspopup="dialog"` + `aria-expanded`
- panel: `role="dialog"` + `aria-label` (또는 labelledby)
- 키보드: Esc 닫기, Tab은 panel 내부 → 닫고 다음으로
- 자동 위치 조정 (viewport 충돌 시 placement 반대로)
- 닫힘 후 트리거에 포커스 복귀

## 8. 코드 예시

```html
<button aria-haspopup="dialog" aria-expanded="true">도움말</button>
<div role="dialog" aria-label="단축키 안내">…</div>
```

## 9. 관련 컴포넌트

- Tooltip — 텍스트 전용 (호버)
- Menu — 옵션 리스트 전용
- Modal — 차단·전체 화면
