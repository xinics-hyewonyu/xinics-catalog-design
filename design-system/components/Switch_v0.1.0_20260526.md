## 1. 정의

두 가지 상태(ON/OFF)를 즉시 전환하는 토글 컨트롤. Checkbox와 달리 폼 제출 없이 즉시 적용되는 설정에 사용.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `track` | ✓ | pill 형 배경 |
| `thumb` | ✓ | 원형 손잡이 |
| `label` | – | 좌·우 라벨 |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| size | `sm` \| `md` |
| state | `off` \| `on` \| `disabled` |

## 4. 토큰 매핑 — state: on

| 속성 | 토큰 | 값 |
|---|---|---|
| track.background (on) | `{alias.action.primary}` | {seed.color.primary} |
| track.background (off) | `{alias.border.default}` | {map.color.neutral.200} |
| thumb.background | `{alias.surface.elevated}` | oklch(1 0 0) |
| track.borderRadius | 999 (pill) | 999px |

## 5. 사용 케이스 (Do)

- 설정 화면의 ON/OFF (알림 켜기/끄기 등)
- 즉시 반영되는 토글 (저장 없이)

## 6. 사용 금지 (Don't)

- 다중 선택 → `Checkbox`
- 폼 제출이 필요한 선택 → `Checkbox`

## 7. 접근성

- `role="switch"` + `aria-checked` 필수
- 색만으로 ON/OFF 구별 금지 (P-06) — 라벨/위치/icon 병행
- 키보드: Space로 토글
- 터치 타겟: 32×32px 이상 확보 (O-09)

## 8. 코드 예시

```html
<button role="switch" aria-checked="true" class="xds-switch">
  <span class="xds-switch__thumb"></span>
</button>
```

## 9. 관련 컴포넌트

- Checkbox — 폼 제출에 포함되는 다중 선택
- ToggleButton — 라벨이 본문에 노출되는 명시 토글
