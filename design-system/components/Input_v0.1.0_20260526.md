## 1. 정의

텍스트·숫자 등 단일 줄 입력을 위한 폼 컨트롤.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `root` | ✓ | 입력 컨테이너 |
| `input` | ✓ | <input> 요소 |
| `iconLeading` | – | 좌측 장식·검색 아이콘 |
| `clear` | – | 입력 지우기 버튼 |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| size | `sm` \| `md` \| `lg` |
| state | `default` \| `focus` \| `disabled` \| `error` |

## 5. 사용 케이스 (Do)

- 단일 줄 텍스트·숫자 입력
- placeholder는 *예시*에만 사용 (라벨 대체 금지)

## 6. 사용 금지 (Don't)

- 여러 줄은 `Textarea`
- 선택지 중 하나는 `Select`
- 라벨 없이 placeholder만 (KWCAG 4.1 위반)

## 7. 접근성

- `<label>` 또는 `aria-label` 필수 (U-03)
- 오류 메시지는 `aria-describedby`로 연결 (U-05)
- 한국어 입력 형식 사전 안내 (U-13)
- 포커스 링 가시성 2px+ (O-03)

## 8. 코드 예시

```html
<label>
  <span>이메일</span>
  <input type="email" class="xds-input" />
</label>
```

## 9. 관련 컴포넌트

- Textarea — 여러 줄 입력
- Select — 선택지 입력
- SearchField — 검색 전용
