## 1. 정의

트리거 요소에 hover·focus 시 짧은 도움말 텍스트를 띄우는 비차단 팝업. 클릭 가능한 액션 없음.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `trigger` | ✓ | 호버·포커스 대상 |
| `bubble` | ✓ | 텍스트 컨테이너 |
| `arrow` | – | 트리거 방향 표시 |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| placement | `top` \| `bottom` \| `left` \| `right` |

## 5. 사용 케이스 (Do)

- 아이콘 버튼의 보조 라벨
- 단축키 안내 (예: "⌘K")

## 6. 사용 금지 (Don't)

- 클릭 가능한 액션 포함 → `Popover`
- 라벨 자체가 없는 컨트롤의 *유일한* 라벨로 사용 (U-03 위반 — 시각 라벨 필요)
- 모바일 hover 의존 (터치 트리거 별도 처리)

## 7. 접근성

- `role="tooltip"` + 트리거의 `aria-describedby`
- Esc로 닫기, 포커스 이동 시 자동 노출
- 텍스트만 (이미지·인터랙티브 금지)
- 대비비: 역상 색 조합도 4.5:1 통과해야 (P-01)

## 8. 코드 예시

```html
<button aria-describedby="tt1">⚙</button>
<span role="tooltip" id="tt1">설정</span>
```
