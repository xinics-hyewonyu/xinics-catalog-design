## 1. 정의

관련 정보를 시각적으로 묶어 표시하는 컨테이너.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `root` | ✓ | 카드 컨테이너 |
| `header` | – | 제목·메타 영역 |
| `body` | ✓ | 본문 영역 |
| `footer` | – | 액션·메타 영역 |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| elevation | `flat` \| `raised` \| `interactive` |

## 4. 토큰 매핑 — elevation: raised

| 속성 | 토큰 | 값 |
|---|---|---|
| background | `{alias.surface.elevated}` | oklch(1 0 0) |
| border | `{alias.border.subtle}` | {map.color.neutral.100} |
| borderRadius | `{map.radius.lg}` | 27px |
| padding | `{map.size.md}` | 16px |

## 5. 사용 케이스 (Do)

- 진도·통계 등 단위 정보 묶음
- 리스트 항목으로 사용 시 `interactive` elevation

## 6. 사용 금지 (Don't)

- 한 화면에 5개 이상의 raised card (시각 혼란)
- card 안에 또 다른 card 중첩

## 7. 접근성

- `interactive` 변형은 키보드 포커스 가능 (`tabindex="0"` + role)
- 헤더가 있으면 의미 단위 구분 (`<h3>` 등)

## 8. 코드 예시

```html
<article class="xds-card xds-card--raised">
  <h3>강의 시청 완료</h3>
  <p>총 8개 중 6개를 완료했어요.</p>
</article>
```
