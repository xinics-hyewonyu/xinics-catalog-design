## 1. 정의

현재 위치까지의 계층 경로를 표시하는 부가 네비게이션. 사용자가 *어디에 있는지* 알리고 상위로 빠르게 이동.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `list` | ✓ | `<ol>` 또는 `<nav>` |
| `item` | ✓ | 개별 경로 항목 |
| `separator` | ✓ | "/"·">"·"›" 등 |
| `current` | ✓ | 현재 페이지(마지막) |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| separator | `/` \| `>` \| `›` |

## 4. 토큰 매핑

| 속성 | 토큰 | 값 |
|---|---|---|
| link.color | `{alias.text.link}` | {seed.color.primary} |
| current.color | `{alias.text.body}` | {map.color.neutral.900} |
| separator.color | `{alias.text.caption}` | {map.color.neutral.700} |

## 5. 사용 케이스 (Do)

- 3단계 이상 깊이의 페이지
- 같은 사이트 안 다른 계층 이동

## 6. 사용 금지 (Don't)

- 1~2단계 얕은 사이트 (불필요)
- 모달·다이얼로그 내부

## 7. 접근성

- `<nav aria-label="브레드크럼">` 래퍼
- 마지막 항목 `aria-current="page"` + 링크 아님
- 구분자는 `aria-hidden` (스크린리더는 의미 없음)

## 8. 코드 예시

```html
<nav aria-label="브레드크럼">
  <ol>
    <li><a href="/">대시보드</a></li>
    <li><a href="/projects">프로젝트</a></li>
    <li aria-current="page">AICMS Admin</li>
  </ol>
</nav>
```

## 9. 관련 컴포넌트

- Navigation — 사이트 전역 네비
- Steps — 순차 진행 (다른 의미)
