## 1. 정의

여러 콘텐츠 패널을 *접고 펼 수 있는* 세로 그룹. FAQ·약관·필터 등 *선택적 노출* 정보에 사용.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `item` | ✓ | 단일 펼침 단위 |
| `trigger` | ✓ | 헤더 (클릭 시 펼침) |
| `panel` | ✓ | 펼쳐지는 콘텐츠 |
| `indicator` | – | ▾ 아이콘 (회전 애니메이션) |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| mode | `single`(1개만 펼침) \| `multiple`(여러 개 동시) |
| variant | `bordered` \| `seamless` |

## 5. 사용 케이스 (Do)

- FAQ — 질문은 항상 보이고 답변은 펼침
- 약관·정책 — 긴 텍스트 부담 감소
- 사이드바 필터 그룹

## 6. 사용 금지 (Don't)

- 항상 필요한 핵심 정보 (숨김 부적절)
- 콘텐츠가 짧아 펼침 가치 없는 경우
- 깊은 중첩 (Accordion 안 Accordion)

## 7. 접근성

- trigger: `<button>` + `aria-expanded`, `aria-controls`
- panel: `role="region"` + `aria-labelledby=trigger`
- 키보드: Tab으로 trigger 진입, Space/Enter로 펼침
- 시각 indicator 회전 + `prefers-reduced-motion` 시 즉시 전환

## 8. 코드 예시

```html
<button aria-expanded="true" aria-controls="p1">개인정보 처리방침</button>
<div id="p1" role="region">수집·이용 항목·보관 기간…</div>
```

## 9. 관련 컴포넌트

- Tabs — 같은 위계 콘텐츠 전환 (한 시점 1개)
- Collapse — 단일 펼침 (그룹 아님)
