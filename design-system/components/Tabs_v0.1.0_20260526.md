## 1. 정의

같은 위계의 콘텐츠 영역을 가로로 전환하는 네비게이션. 한 시점에 1개 탭만 활성.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `list` | ✓ | 탭 라벨 행 (role="tablist") |
| `tab` | ✓ | 개별 탭 (role="tab") |
| `panel` | ✓ | 활성 탭의 콘텐츠 (role="tabpanel") |
| `indicator` | – | 활성 탭 강조 라인 |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| style | `underline` \| `pill` \| `enclosed` |
| size | `sm` \| `md` |

## 4. 토큰 매핑 — style: underline, state: active

| 속성 | 토큰 |
|---|---|
| indicator.background | `{alias.action.primary}` |
| tab.color (active) | `{alias.action.primary}` |
| tab.color (inactive) | `{alias.text.caption}` |
| list.borderBottom | `{alias.border.subtle}` |

## 5. 사용 케이스 (Do)

- 같은 위계의 콘텐츠 2~7개 전환
- 페이지 내 섹션 빠른 이동

## 6. 사용 금지 (Don't)

- 탭 수 8개 초과 (가로 스크롤·접힘 → `Select`나 `Sidebar`)
- 탭이 페이지 전환 (라우팅) → 별도 네비게이션

## 7. 접근성

- `role="tablist" / "tab" / "tabpanel"` 필수 (WAI-ARIA APG)
- 키보드: ←→ 로 활성 탭 이동, Home/End 지원
- 활성 탭의 `aria-selected="true"`, 패널의 `aria-labelledby`
- 색만으로 활성 구분 금지 (P-06) — underline·굵기 병행

## 8. 코드 예시

```html
<div role="tablist">
  <button role="tab" aria-selected="true">개요</button>
  <button role="tab">토큰</button>
</div>
<section role="tabpanel">…</section>
```

## 9. 관련 컴포넌트

- Accordion — 세로 펼침 그룹
- Stepper — 순차 진행 단계
- SegmentedControl — 라디오의 시각적 압축형
