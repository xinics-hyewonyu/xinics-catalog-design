## 1. 정의

비결정적(unbounded) 대기를 시각화하는 회전 인디케이터. 시간 예측이 어려운 짧은 로딩에 사용.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `ring` | ✓ | 회전하는 원 (3/4 호) |
| `label` | – | 보조 텍스트 (선택) |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| size | `sm`(16) \| `md`(20) \| `lg`(28) |
| tone | `primary` \| `muted` |

## 4. 토큰 매핑

| 속성 | 토큰 | 값 |
|---|---|---|
| ring.borderColor | `{alias.border.default}` | {map.color.neutral.200} |
| ring.activeColor | `{alias.action.primary}` | {seed.color.primary} |

## 5. 사용 케이스 (Do)

- 1~10초 사이 짧은 비동기 작업
- 버튼 내부 로딩 (Submit 중)

## 6. 사용 금지 (Don't)

- 시간 예측 가능 작업 → `Progress`
- 페이지 전체 로딩 시 단독으로 → `Skeleton`이 더 친절
- 10초 이상 지속 → 진행률·취소 가능 패턴으로 교체

## 7. 접근성

- `role="status"` + `aria-label="로딩 중"`
- `prefers-reduced-motion`: 회전 → 펄스(opacity) 대체
- 색만으로 의미 전달 금지 — 텍스트 라벨 권장

## 8. 코드 예시

```html
<span class="xds-spinner" role="status" aria-label="로딩 중"></span>
```

## 9. 관련 컴포넌트

- Progress — 진행률이 있는 결정적 작업
- Skeleton — 페이지·카드 단위 placeholder
