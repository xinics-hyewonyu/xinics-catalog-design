## 1. 정의

콘텐츠 영역을 시각적으로 구분하는 *얇은 선*. 단순 가로/세로 선 또는 가운데 라벨 포함 형태.

## 2. Anatomy

| 슬롯 | 필수 | 설명 |
|---|---|---|
| `line` | ✓ | 구분선 |
| `label` | – | 가운데 텍스트 ("섹션 구분" 등) |

## 3. Variants

| 차원 | 옵션 |
|---|---|
| orientation | `horizontal` \| `vertical` |
| variant | `plain` \| `dashed` \| `with-label` |

## 4. 토큰 매핑

| 속성 | 토큰 | 값 |
|---|---|---|
| line.color | `{alias.border.subtle}` | {map.color.neutral.100} |
| line.thickness | 1px | 1px |
| label.color | `{alias.text.caption}` | {map.color.neutral.700} |

## 5. 사용 케이스 (Do)

- 메뉴 그룹 사이 구분
- 폼 섹션 구분 ("기본 정보" / "보안")
- 가로 콘텐츠 안 항목 분리 (vertical)

## 6. 사용 금지 (Don't)

- 헤딩으로 이미 구분된 섹션에 중복 추가
- 너무 짧거나(< 1em) 너무 두꺼운 (3px+) 선

## 7. 접근성

- `role="separator"` (기본 `<hr>`은 자동)
- 라벨이 있으면 `aria-label`
- 장식 전용이면 `aria-hidden`

## 8. 코드 예시

```html
<hr class="xds-divider" />
<div role="separator" aria-label="섹션 구분"><span>섹션 구분</span></div>
```

## 9. 관련 컴포넌트

- Section — 시맨틱 그룹화
- Stack — gap만으로 구분
